// Objetivo
Comprender cómo está organizado internamente un sistema computacional: procesador, memoria, buses y la relación entre hardware y software a nivel arquitectónico.
// Metodología
Cada semana incluye una página de referencia con los 12 temas desarrollados, diagramas SVG integrados y un taller por grupos con clave de acceso.
// Evaluación
Talleres grupales por semana, práctica de aula y exposición. El taller de grupos requiere clave de acceso asignada por el docente al inicio de cada sesión.
